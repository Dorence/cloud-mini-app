# About The project 
## Aim
本项目是为了完善 **HUST** 社联小程序的功能而发起的
## 场地借用系统
### 需求
**主要功能**
- 空闲状况公示
- 前端交表->秘书部审批（通过/不通过 + 备注）
- 审批通过后申请人能打印凭证
- 后台能导出表格存档

要求：
- 表格有附件上传需求
- 填表者/管理者均要登陆

_Extra_ 
- 管理者能取消借用审批
- 填表过程进度保存
